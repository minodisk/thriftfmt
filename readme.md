# thriftfmt

A formatter for Apache thrift.

## Installation

### Binary

Download the appropriate version for your platform from the release page and place the binary in your `$PATH`.

### Go Get

```bash
go get -u github.com/minodisk/thriftfmt
```

## Usage

```bash
thriftfmt --help
```

## Support

### Syntax

#### Head

- [ ] namespace
- [ ] enum
- [ ] const
- [ ] struct

#### Definition

- [ ] service
- [ ] function

##### Field

- [ ] doc
- [ ] annotation

### Command

- [ ] Display diffs
- [ ] Report all errors
- [ ] List files
- [ ] Write result to file
